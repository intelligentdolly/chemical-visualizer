# Chemical Visualizer Web Application

This project is a full-stack web application built using:

- **Django (Backend API)**
- **React.js (Frontend UI)**
- **SQLite3 (Database)**
- **Material UI (React UI Library)**

This application allows users to **add, view, and visualize chemicals** with details like:
- Name  
- Formula  
- Molecular Weight  
- Description
